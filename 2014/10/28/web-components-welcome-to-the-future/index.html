Web components – welcome to the future

Zapraszamy do posłuchania o nowym trendzie w świecie JavaScript – Web Components.

Kilka słów o prezentacji i prelegencie:

Web Components to zasadnicza zmiana sposobu programowania aplikacji internetowych. Oferują nam one narzędzia, które są potrzebne aby uczynić pracę developera bardziej produktywną oraz dają dostęp do możliwości zarezerwowanych wcześniej tylko dla przeglądarek internetowych.

Karol Fabjańczuk

Front-end developer w firmie Lemon Demon. Zainteresowany open source, fotografią lotniczą oraz najnowszymi technologiami.
